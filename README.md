# Moirai Trading Buddy

Moirai Trading Buddy is a simple HTML-based web tool designed to assist both beginner and experienced Forex traders.  
It provides quick reference to trading pairs, trading session times, helpful charting resources, and strategy guidance.

- Navigation menu for quick section access
- Table of recommended trading pairs with opening times
- External resource links for news and charting
- Simple trading strategy reference table
- Fully responsive and easy to expand
